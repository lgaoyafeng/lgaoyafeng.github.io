3. linux 安装git

1、在进行安装前，需要安装一些依赖的包

yum install curl-devel expat-devel gettext-devel openssl-devel zlib-devel gcc perl-ExtUtils-MakeMaker

下载git源码并解压
目前最新版本下载地址：https://github.com/git/git/releases/tag/v2.11.0
解压 tar zxvf git-2.11.0.tar.gz

cd git-2.11.0

编译安装
make prefix=/usr/local/git all
make prefix=/usr/local/git install

查看git
whereis git
git –version

配置环境变量
vim /etc/profile
加入export PATH=$PATH:/usr/local/git/bin
生效配置文件 source /etc/profile

4.linux安装jenkins(rpm安装,还可以安装docker版本,war版本)

下载jenkins: https://pkg.jenkins.io/redhat-stable/

安装下载好的 rpm -ivh jenkins-2.213-1.1.noarch.rpm

更改jenkins的配置文件 vim /etc/sysconfig/jenkins

目前只需更改服务端口

vi /etc/sysconfig/jenkins

service jenkins start 启动jenkins服务

根据页面提示位置/var/lib/jenkins/secrets/initialAdminPassword，获取初始密码

初始化并配置jenkins ,

官方下载插件慢 更新下载地址

cd {你的Jenkins工作目录}/updates #进入更新配置位置

sed -i ‘s/http://updates.jenkins-ci.org/download/https://mirrors.tuna.tsinghua.edu.cn/jenkins/g’ default.json && sed -i ‘s/http://www.google.com/https:\/\/www.baidu.com/g' default.json

之后进行重启,然后再下载官方推荐的插件, 记得下载本地化的插件 local 改为中文

然后在 系统管理=>全局工具配置=>中配置 jdk,maven ,git

​ jdk, maven 路径是项目的完整路径名, git 的路径需要先执行 which git,然后将打印出的路径配置为git 的home路径, 保存退出
